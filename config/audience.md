## 1. Who is the precise buyer? (persona)

**Primary Buyer — "资本闲置型高净值个人"（Idle Capital Professional）**

- **年龄段：** 35–55 岁
- **职业：** 中小企业主（SME owner）、专业人士（律师、工程师、医生、会计师）、已在职高薪打工人、二代企业接班人
- **收入 / 资产水平：** 家庭月收入 RM 15,000 以上；名下有定期存款（FD）、产业、或股票投资组合；有可支配资金但对其回报不满意
- **交易经历：** 有过股票（Bursa / 港股 / 美股）或黄金投资经历；听说过外汇 / 期货但从未系统学习；不是零基础小白，不需要被教"什么是 K 线"
- **地域：** 主要为西马（Klang Valley、槟城、新山）；其次为东马及新加坡华人社区
- **语言：** 主力为华语（简体中文书写，口语马华腔调）；英文财经术语自然混用（Prop Firm、funded account、SOP、risk management 等）；不使用马来语或英文为主的媒体
- **平台行为：** Facebook 主力用户；活跃于华人理财 / 投资讨论群；会点进"免费线上分享会"但对硬销售警觉性高

**次要受众 — "有经验但亏损过的交易员"（Battle-Scarred Trader）**

- 35–50 岁，曾自己做外汇 / 股票，输过钱
- 不缺资本，缺的是一套可复制的系统（SOP）
- 对"又一个骗局"高度警觉，需要被"内行人"说服

## 2. Their situation / pain (in their own words)

以下为真实买家常说的话（来自学员问卷及销售团队反馈）：

> "钱放 FD 里，利息根本追不上通货膨胀。"

> "我买股票也不是没买过，但太被动了——公司出事才知道，已经来不及了。"

> "想做外汇，又怕被骗。市面上教的不知道哪个真、哪个假。"

> "没时间盯盘。我要打理生意 / 要上班，哪有时间每天开着图表看几个小时？"

> "用自己的钱练，输了就真的没了。没有人教我怎么控风险。"

> "之前跟单、买信号群，最后都是一场空。我要自己会，不要依赖别人。"

> "我不要变成全职交易员，我只是要一个稳定的副收入来源。"

**核心触发情境：**
- 刚把一笔定存到期的钱取出，不知道放哪里
- 生意收入开始停滞，意识到不能只靠主业
- 朋友圈有人做 Prop Firm 赚钱，产生了好奇和危机感
- 市场波动大，旧有的理财方式感觉越来越不够用

## 3. Their desire / outcome

**他们想要的转变（合规表述框架）：**

不是"一夜暴富"——是"学会一套自己可以操作的系统，让钱产生额外的流动性"。

具体来说：

- **掌握一套可复制的 SOP**：看到信号就知道买涨还是买跌，不靠感觉、不靠消息
- **用时间换效率，不是用时间换钱**：1 分钟内完成决策，不占用本业时间
- **用 Prop Firm 机制降低自有资金风险**：用经过审核的资金授权，而不是把自己的本金直接暴露在市场里
- **建立一个和主业平行运作的收入来源**：不是替代工作或生意，而是让资本不再闲置
- **独立判断，不依赖群、不跟单**：学完之后自己会做，不需要再买信号

**合规框架：**
- 所有结果表述为"学习技能 / 掌握方法"，不承诺具体盈利数字
- 学员案例以"他学会了之后开始…"表述，而非"他一个月赚了 RM X"
- Prop Firm 收益框架为"有机会透过资金授权参与市场"，不说"无风险赚美金"

## 4. Objections / disqualifiers

### 🚫 这不是我们的买家（排斥对象）

- 月收入低于 RM 5,000、没有任何可支配资产的人（经济压力导致决策情绪化，不适合学交易）
- 寻找"跟单群"或"保证盈利信号"的人（期望错位，无法从系统化教育中受益）
- 完全零基础、从未接触任何投资产品的人（需要大量前期教育，不是我们课程的起点）
- 想"快速翻身""赌一把"的赌徒心态者（会对品牌造成损害）
- 年龄低于 25 岁、收入不稳定的学生（购买力不足，客单价匹配度低）

### ⚡ 买家的主要异议（文案需预先化解）

1. **"这是不是诈骗？"** → 用 Prop Firm 的国际机构背景（FTMO、The 5%ers 等）和 Soo Cheng 老师的可查证资历化解
2. **"我没时间学"** → 强调"1 分钟完成决策"、非全职交易、配合生意 / 上班时间
3. **"我之前被骗过 / 亏过"** → 验证痛苦，重新框架为"那是因为你没有系统，不是你不适合"
4. **"为什么要用 Prop Firm，我自己有钱"** → 框架为"有钱更要保住本金"，Prop Firm 是专业交易员的风险管理工具，不是因为穷才用
5. **"免费分享会是不是会强迫销售？"** → 强调内容价值优先，CTA 以"来学东西"为主轴

## 6. Voice & language register

### 核心语气定位

**有质感的同行对话（Peer-to-Peer）**——Soo Cheng 老师是你信任的、懂行的朋友，不是在台上俯视你的"讲师"。

广告文案由团队第三人称撰写，介绍 Soo Cheng 老师的洞察，语气为：见过世面的人，在和另一个见过世面的人说话。

### 语言混用规则

| 用华语 | 保留英文（不翻译） |
|--------|------------------|
| 本钱、资金、钱 | Prop Firm |
| 做交易 | SOP、formula |
| 买涨 / 买跌 | funded account |
| 控风险 | challenge |
| 开单 / 平仓 | risk management |
| 盯盘 | entry / exit |

### 句式偏好

- **排比结构** 制造节奏感：「盯盘盯的是情绪，不是机会。越盯越焦虑，越焦虑越乱做决定。」
- **反问句** 制造共鸣：「如果 1 分钟就能完成交易，你还会盯盘 8 小时吗？」
- **「」书名号** 框住核心方法名：「1 分钟极速交易策略」
- **短段落 + 空行**：手机端阅读优先，每个想法独立一段

### Emoji 规则

- 📍 用于课程内容 bullet points
- 👇 用于 CTA
- 🔴 用于标题建议
- **禁止** 使用：❌✅🎓🔑💰💸🚀 等表情——显得廉价且不符合品牌调性

### 禁用词汇

| 禁止 | 原因 |
|------|------|
| 诈骗、骗局、赌 | Facebook 金融广告政策风险 |
| 一定赚、保证盈利、无风险 | 金融监管合规 |
| 月入 RM X、年化 X% | 收益承诺，不合规 |
| 破解、套利密码、秘籍 | 显得不专业，与品牌调性冲突 |
| 啦、咩、leh、酱 | 过于随意，与 peer-level 定位不符 |
| 小白、新手、穷人 | 触犯受众定位（对方是有钱、有经验的人） |
| 投资者、启动资金 | 大陆正式用语，马来西亚华人不自然 |

## 7. Compliance notes (financial-education product)

- **无盈利承诺**：所有文案不得出现具体收益数字、预期回报、或任何形式的"保证"字眼
- **教育定位**：产品为金融教育课程，不是投资建议或资产管理服务
- **风险意识**：在适当位置加入"交易涉及风险，学员需自行判断"相关表述（尤其在落地页和课程介绍）；广告文案无需每句带，但整体不能暗示"零风险"
- **Prop Firm 框架**：Prop Firm 是独立第三方机构，Soo Cheng 老师教的是通过其评核的方法，不是直接提供资金或保证通过
- **学员案例**：只展示"学员通过了 challenge / 获得资金授权"等技能验证结果，不展示具体盈利金额
- **"免费"分享会**：可说"免费报名"，不可说"价值 RM X，今天免费"（夸大价值陷阱）
- **Facebook 广告政策额外限制**：避免使用"外汇"（Forex）直接作为广告主关键词——用"期货交易教育"、"交易策略"替代；避免在广告图片中展示模拟盈利曲线
- **MTC 内部附加规则**：文案不得针对受众的财务困境施压（如"你现在的工作养不活你"类表述）；受众定位为资本充裕者，不以恐惧贫穷为驱动
